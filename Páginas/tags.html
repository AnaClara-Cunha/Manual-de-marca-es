Tags e informações mais básicas

  • Títulos: <h1> a <h6>, a depender do nível
  • Parágrafos: <p>
  • Quebra de linha:<br>
  • Linha horizontal na página: <hr>
  • Comentários: <!-- -->
  • Símbolos: &+nome do símbolo. Exemplo: ® (&+reg)
  • Emojis: &#x + número do emoji (pegar o código no emojipedia). Exemplo: 😊 (&#x + U+1F60A, tirando o U+)
  • Tab + shift: dá o espaçamento que o Tab dá, mas para a esquerda, como recuo. Se selecionadas as linhas desejadas, é possível recuar todas de uma vez.
  • Para fazer uma pesquisa de ferramentas/abrir um atalho no VS Code, basta clicar: CTRL + SHIFT + P. Para adicionar as tags sem precisar de escrever no texto, basta marcar a parte desejada, digitar o comando anterior e pesquisar por ABB ("envolver com abreviação"), depois digitar a tag sem os símbolos maior e menor que.
  • DICA: Para selecionar e escrever em diferentes lugares ao mesmo tempo, utilizar ALT. Só pressionar a tecla e selecionar os lugares desejados.

Principais formatações

  • Negrito/Destaque:
    • Forma semântica: <strong>
    • Forma não semântica: <b>
  • Itálico/Ênfase
    • Forma semântica: <em>
    • Forma não semântica: <i>
  • Texto Marcado: <mark>
  • Texto Grande: <big> - OBSOLETO*
    *Tags que ficam vermelhas, como a <center>e <bug> não devem ser usadas! Estão obsoletas, apesar de funcionarem, ou seja, podem deixar de funcionar a qualquer momento.
  • Texto Pequeno: <small> (não se enquadra como obsoleto, porque letras pequenas podem ser pequenas por questões além da forma, como as de um contrato)
  • Texto deletado: <del> exemplo
  • Texto inserido: <ins> exemplo
    Existe também o sublinhado com a tag <u>(não semântico).

  • Texto Sobrescrito: <sup> exemplo
  • Texto Subscrito: <sub> exemplo
  • Fonte monoespaçada: <code>
    Exemplo: Console.WriteLine("Olá, Mundo!");
    • Para fazer um código com indentação: adicionar <pre> antes do <code>
      Exemplo: 
                              
                        num = int(input('Digite um número'))
                        if num % 2 == 0;
                                print(f'O número {num} é PAR')
                        else:
                                print(f 'O número {num} é IMPAR')
                        print ('Fim do programa')
  • Citações simples: <pre> (mesma linha). Exemplo: Oi, como vai?
  • Citações grandes: <bloquote>. Exemplo: Ele disse vai, eu disse vou
    É possível somente adicionar as aspas, mas colocar quote leva significado.
  • Abreviações: <abbr> (coloca o significado da abreviação na palavra, quando o cursor passa por ela no texto).
    Exemplo: HTML
  • [Inútil] Texto invertido: <bdo>
    Exemplo: Estou aprendendo a criar coisas em HTML.
