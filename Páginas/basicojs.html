Básico de JavaScript

  1. Ponto e vírgula ao final das linhas é opcional
  2. Variáveis não precisam ter seu tipo definido no momento de sua declaração
  3. O JS considera a variável como string autometicamente. Deve-se fazer a conversão para os demais formatos. A saber:
    1. Para número é possível utilizar "Number.parseInt()", "Number.parseFloat()" ou simplesmente "Number()" (neste último, o JS identifica automaticamente o tipo de número utilizado.
    2. Também é possível converter sem utilizar o início Number, somente com "parseInt()".
    3. Para converter a string, pode-se utilizar "String()" ou "variavel.toString()"
  4. Window.alert faz aparecer um pop-up na tela.
  5. Window.prompt faz aparecer um pop-up que precisa ser preenchido na tela.
  6. variavel.length (atributo, não tem parênteses ao final: informa quantos caracteres a string tem.
  7. variavel.toUpprtCase(): transforma as letras em maíusculas.
  8. variavel.toLowerCase(): transforma as letras em minúsculas.

    9. Tipos de comando

    Existem diferentes tipos de comando para juntar o que se quer dizer e o valor das variáveis.

    1. É possível fazer utilizando concatenação: "Obrigado por se registrar" + nome "!"
    2. Também é possível fazer a mesma coisa utilizando Template String (que deve ser escrito entre crases e a variável fica dentro de um place holder (${})): `Obrigado por se registrar ${nome}!`

Operadores

    Aritméticos
  • Mais: +
  • Menos: -
  • Vezes: *
  • Dividido: /
  • Resto da divisão: %
  • Potência: **
    De atribuição
  • Atribuição simples/recebe: =
  • Auto-atribuição: variável recebe o valor dela mesmo com alguma operação: += | -= | *= | /= | **= | %=
    Relacionais
  • 5 == 5 -> True
  • 5 == '5' -> JS considera true, porque o sinal de igualdade não testa o tipo
  • 5 === '5' -> False. 3 iguais avalia se é identico, então leva em conta o tipo.
  • Desigual restrito: !==, leva em conta o tipo
    Lógicos
  • Negação: ! -> operador unário, após ele se coloca true, false ou uma expressão que retorne algum desses valores
  • Conjunção (e): && -> operador binário, tem 2 condições e segue a lógica
  • Disjunção (ou): || -> operador binário, tem 2 condições e segue a lógica
  • Operadores lógicos têm ordem de precedência: não (!), e (&&), ou(||)
    Ternário
  • teste ? true : false -> tem 3 partes. Há um teste lógico, se o resultado do teste for verdadeiro, aparecerá o que está ao lado do "?", se for falso, após o "":".
    Exemplo: média >= 7.0 ? "Aprovado" : "Reprovado"

O que é DOM?

Document Object Model - facilitador dentro do navegador e do HTML5 para dar acesso aos elementos dentro do JS. Sem ele, haveria muita dificuldade para acessar elementos da página.

InnerText -> texto que está dentro do texto, sem formatação

InnerHTML -> pega toda a formatação e p texto
