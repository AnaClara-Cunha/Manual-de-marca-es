Links

Usa-se "<a + href = ""></a>" para links em geral.

  • Adiciona-se "rel="prev" após o href para indicar que o link volta para alguma página.
  • Adiciona-se "rel="next" após o href para indicar que o link segue para alguma página.
  • Adiciona-se "rel="nofollow" após o href para indicar que o link segue para alguma página que não deve ser indexada junto ao site. De alguém externo, como uma publicidade.
  • Adiciona-se "target="_blank" após o href se desejar que o link abra em outra página
  • Adiciona-se "target="_self" após o href se desejar que o link abra na mesma página
    • Apesar de automático, adicionar o target facilita para os buscadores a identificação do conteúdo do seu site e a organização deste.

Links internos de outras pastas

  • É preciso indicar a pasta: <href="pasta/nome_do_arquivo>
  • Se for um item em uma pasta anterior à utilizada, deve-se usar <href="../arquivo.html">.
    O "../" volta uma pasta.
